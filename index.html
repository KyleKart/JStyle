JStyle

A fresh take on styling with JavaScript

What is JStyle?

JStyle lets you write your styles using JavaScript syntax.
This website’s styles are even created with JStyle!

JStyle may not be as powerful as CSS, but it’s a creative way to style your pages.
JStyle is not intended as a serious CSS replacement!

How do I use JStyle?

To use JStyle you need to add this to your site:
<script src="https://kylekart.github.io/JStyle/jstyle.js"></script>

And then use a jstyle tag like this:

<jstyle> h1 = { color: "#FFFFFF", fontWeight: "600" }; h1.backgroundColor = "#1a1a1a"; $someID = { padding: "10px", backgroundColor: "#333333", color: "#FFD700" }; _aClass = { padding: "10px", color: "#FF8100", textDecoration: "underline" }; </jstyle>
